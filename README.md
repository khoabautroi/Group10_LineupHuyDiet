# Optimization Algorithms: From Classical to Nature-Inspired

This project provides a comprehensive study and implementation of various **search and optimization algorithms**, ranging from **classical graph search methods** to advanced **nature-inspired metaheuristic algorithms**.

The repository explores algorithmic principles, implementation strategies, and experimental evaluation of optimization methods widely used in **Artificial Intelligence, Operations Research, and Computational Optimization**.

---

## 📌 Project Overview

Optimization algorithms play a crucial role in solving complex problems where traditional deterministic approaches become inefficient.  
This project investigates different optimization paradigms and analyzes how they perform on both **continuous** and **discrete optimization problems**.

The study includes theoretical explanations, algorithm implementations, and experimental comparisons across multiple benchmark functions.

The repository covers algorithms such as:

- A* Search
- Genetic Algorithms (GA)
- Differential Evolution (DE)
- Particle Swarm Optimization (PSO)
- Ant Colony Optimization (ACO)
- Artificial Bee Colony (ABC)
- Firefly Algorithm (FA)
- Cuckoo Search (CS)
- Teaching-Learning-Based Optimization (TLBO)

## 🚀 Key Experiments

### Continuous Optimization
Algorithms are evaluated on several standard benchmark functions commonly used in optimization research.

Examples include:

- Sphere Function
- Rastrigin Function
- Ackley Function
- Griewank Function

These functions allow analysis of algorithm behavior in both **unimodal and multimodal landscapes**.

---

### Discrete Optimization

Two classical combinatorial optimization problems are included:

**Traveling Salesman Problem (TSP)**  
- Instance with **30 cities**
- Evaluates routing optimization capabilities.

**Knapsack Problem**
- Dataset with **20 items**
- Evaluates binary decision optimization.
