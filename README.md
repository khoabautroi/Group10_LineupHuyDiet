# Group10_LineupHuyDiet
Optimization Algorithms: From Classical to Nature-Inspired
This project provides a comprehensive study and implementation of various search and optimization algorithms, ranging from classical graph search to advanced nature-inspired metaheuristics.

📌 Project Overview
The repository explores different optimization paradigms, evaluating their performance on both continuous and discrete optimization problems. It covers the taxonomy, mechanisms, and comparative analysis of algorithms such as A*, Genetic Algorithms, Particle Swarm Optimization, and more.

Detailed analysis of Hill Climbing and Simulated Annealing.

Chapter 4: Evolution-Based Algorithms

Genetic Algorithm (GA) and Differential Evolution (DE).

Chapter 5: Swarm & Biology-Based Algorithms

ACO, PSO, ABC, Firefly Algorithm, and Cuckoo Search.
